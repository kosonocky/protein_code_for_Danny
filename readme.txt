
Installation:

Linux Bash:
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt

Cmd:
$ python3 -m venv venv
$ venv\Scripts\activate.bat
$ pip install -r requirements.txt

Git Bash:
$ python3 -m venv venv
$ source venv/Scripts/activate
$ pip install -r requirements.txt
